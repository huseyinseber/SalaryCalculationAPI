# SalaryCalculationAPI

 `Veritabanı ve Procedure.txt` dosyasıyla veritabanı tabloları ve procedure'ler kolayca oluşturulabilir. Ayrıca uygulamada **AutoMapper** ve **Serilog** entegrasyonu mevcuttur.

## Özellikler
- `Veritabanı ve Procedure.txt` ile veritabanı (tables & procedures) oluşturma.
- AutoMapper ile DTO ↔ Entity eşlemeleri.
- Serilog ile gelişmiş uygulama loglama.
- .NET (örn. .NET Core / .NET 8) tabanlı yapı (uygulama tipine göre değiştirilebilir).
